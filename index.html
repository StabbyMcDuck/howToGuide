How-To Guide for the Ravelry API

by Regina Imhoff

About this guide:

This is a how-to guide for Ravelry's API.

The API is an easy way to access data from Ravelry, including user data (name, location, age, birthday, etc.), project information (yarns used, yardage, sizes, design features, colors, category, etc.) and forum post information.
