Wrapping Up

...now where can I go?

This tutorial was a simple example of what you can do with the Ravelry API. In addition to using other languages(Python?) and technologies (mobile apps?), you can do a wide variety of searches, comparisons, and collections of the vast data available to programmers through the Ravelry API. Some of those examples would be:

  • Colors
  • Patterns
  • Users
  • Lots more!

You can always look at all of the different JSON calls available by checking on www.ravelry.com/api
If can think of a new JSON call that would be helpful, you can always post on the Ravelry API Group forums.
Developers there are nice and very helpful!
